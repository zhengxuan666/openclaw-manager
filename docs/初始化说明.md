# OpenClaw Manager 初始化说明

本文档用于快速完成本项目的本地初始化、开发启动与 Web 部署启动。

## 1. 环境要求

- Node.js >= 18
- Rust >= 1.70
- npm（或 pnpm）

## 2. 安装依赖

在项目根目录执行：

```bash
npm install
```

## 3. Tauri 开发环境（桌面版）

```bash
npm run tauri:dev
```

## 4. Web 一体化部署（推荐）

### 4.1 构建

```bash
npm run web:build
```

### 4.2 启动（守护进程）

```bash
bash scripts/1panel_web_start.sh
```

### 4.3 调试启动（前台打印日志）

```bash
bash scripts/1panel_web_start.sh --no-daemon
```

### 4.4 停止

```bash
bash scripts/1panel_web_stop.sh
```

## 5. 构建发布版本

```bash
npm run tauri:build
```

## 6. 关键目录说明

- `src/`：React + TypeScript 前端代码
- `src-tauri/`：Tauri / Rust 后端代码
- `public/`：静态资源
- `dist/`：前端构建产物
- `scripts/`：部署启动脚本（含 1Panel 启停）

## 7. 常见问题（简版）

- macOS 提示“已损坏，无法打开”：可使用 `xattr -cr` 去除隔离属性（详见 `README.md`）。
- 依赖安装失败：确认 Node.js、Rust 版本满足要求。
- 启动报 `cargo: not found`：说明运行环境缺少 Rust 工具链。

## 8. 建议工作流

1. 拉取最新代码
2. 安装依赖
3. Web 部署执行 `npm run web:build`
4. 使用 `scripts/1panel_web_start.sh` 启动
5. 查看日志确认健康状态
